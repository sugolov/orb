# orb-frontend

Frontend for the orb shell — currently a single self-contained HTML file
(`orb-shell.html`) that runs in any modern browser. See [`../ARCH.md`](../ARCH.md)
§3 for the code map and §4.7 for the planned backend wiring.

## Run

```sh
npm run dev
# or: python3 -m http.server 5173
```

Then open <http://localhost:5173/orb-shell.html>.
